Hafta 1: C Programlama Temelleri
CLion IDE kullanılarak C dilinde temel giriş-çıkış işlemleri yapıldı.Türkçe karakter desteği eklenerek "İletişim ve Sunum Teknikleri Dersine Hoşgeldiniz" mesajı yazdırıldı.
Hafta 2: Java Swing Arayüz Tasarımı
NetBeans IDE kullanılarak Java Swing kütüphanesiyle arayüzler tasarlandı.Tasarlanan arayüzler Hesap Makinesi, Alışveriş Sepeti, Öğrenci Kayıt Sistemidir.
Hafta 3: C# ve .NET Form Uygulamaları
Microsoft Visual Studio ortamında C# dili ile masaüstü uygulamaları geliştirildi. Günlük planlama için Ajanda ve Telefon Rehberi uygulamalarının arayüzleri tasarlandı.
Hafta 4: Bootstrap ve HTML Tabanlı Web Uygulaması
Notepad++ editörü üzerinde Bootstrap framework altyapısı kullanılarak dinamik bir web arayüzü geliştirildi. Örnek bir şablon esas alınarak; ana, giriş-çıkış ve hata sayfaları ders içeriklerine uygun şekilde yeniden düzenlendi. Bu çalışma ile teknik bilgilerin bir web sitesi düzeninde derli toplu sunulması sağlandı.
